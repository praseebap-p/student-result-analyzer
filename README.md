# Student Result Analyzer

This project analyzes student performance using Python.

## Features
- Calculates average marks
- Identifies topper
- Pass/Fail classification
- Data visualization using bar chart

## Tech Stack
- Python
- Pandas
- Matplotlib

## Project Files
- main.py
- students.csv

## How to Run
1. Install dependencies:
   pip install pandas matplotlib  
2. Run:
   python main.py  

## Output
Displays analysis in terminal and graph visualization.
